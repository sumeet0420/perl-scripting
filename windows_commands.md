##Some useful windows commands
###Echo something
`echo hello`

###Type tye contents of the file`
`type filename`

###List the directories of the file`
`dir`

###Find a text from a file`
`find "string"` filename

###Find the number of matches from a file
`find \C "string" filename`

###Compare two data
`comp [data1] [data2]`

###Compare two files
`fc file1 file2`
